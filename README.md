# kata-pencil-durability
Kata to simulate wearing down an ordinary pencil writing letters

I wrote my kata in Python 3.7.4.

From the kata-pencil-durability directory, run "python src//pencil.py".
For tests, run "python -m unittest tests.test_pencil" from the
kata-pencil-durability directory.

Test classes are:
TestCreatePencil
TestPencilWrite
TestPencilPointDurabilityCost
TestPencilSharpen
TestEraser
TestEraserDurability
TestEditing
TestBuildPencil
TestUsePencil
